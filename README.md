### leftchildrenMap
基于WebGIS的留守儿童求助信息共享平台


涉及技术：
- SpringMVC + Mybatis
- 数据库采用 MySQL
- 前端 Bootstrap + JQuery
- 地图及图表  百度地图 + Echarts
